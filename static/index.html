בוט יומן - הצטרפות אוטומטית

איך מגדירים

1. צור/השתמש בחשבון Gmail ייעודי למערכת.
2. הפעל אימות דו-שלבי, ואז צור App Password (myaccount.google.com → Security → App passwords) והדבק אותו למעלה.
3. ודא ש-IMAP מופעל ב-Gmail (Settings → Forwarding and POP/IMAP).
4. הזמן את כתובת המייל הזו כמשתתף בכל פגישה ביומן - והמערכת תצטרף ותקליט אוטומטית.
חשוב: כדי שההצטרפות האוטומטית תעבוד, יישום הרקע (הפעל_ברקע) חייב לרוץ והמחשב דלוק ומחובר.
