# מערכת סיכום פגישות (Summery System)

מערכת שמקליטה, מתמללת ומסכמת פגישות — בהשראת Timeless.
תמלול מקומי בעברית (Whisper) + סיכום חכם עם Gemini.

## דרישות מותקנות
- Python 3.12, ffmpeg (הותקנו דרך winget)
- סביבה וירטואלית ב-`.venv/`
- מפתח Gemini בקובץ `.env`

## איך מריצים

### 🖥️ אפליקציית דסקטופ (מומלץ)
לחץ פעמיים על **`פתח_אפליקציה.bat`**, או בטרמינל:
```powershell
.\.venv\Scripts\python.exe -m src.gui
```
חלון עם כפתור הקלטה, בחירת מקור, ותצוגת תמלול + סיכום בעברית.

### 🤖 זיהוי פגישות אוטומטי (כמו Timeless) — רץ ברקע
**רץ אוטומטית עם הדלקת המחשב** (הוגדר קיצור ב-Startup של Windows).
אייקון 🎙️ סגול מופיע **ליד השעון** במגש המערכת כשהמערכת פעילה.

- כשמתחילה פגישת Zoom/Teams/Meet/Webex → קופץ חלון "🔴 התחל הקלטה".
- לחיצה אחת מקליטה; "⏹️ סיים וסכם" → תמלול + סיכום נפתחים אוטומטית.
- **שליטה:** קליק ימני על אייקון המגש → הקלטה ידנית / תיקיית סיכומים / יציאה.

**להפעלה ידנית עכשיו (בלי ריסטרט):** לחץ פעמיים על **`הפעל_ברקע.vbs`**.
**לביטול הפעלה אוטומטית:** מחק את הקיצור מתיקיית `shell:startup`.

### הקלטה חיה בטרמינל (חלופי)
לחץ פעמיים על **`הקלט_פגישה.bat`**, או בטרמינל:
```powershell
.\.venv\Scripts\python.exe -m src.app_cli            # מיקרופון בלבד
.\.venv\Scripts\python.exe -m src.app_cli --system   # + קול מערכת (Zoom/Meet)
```
לוחצים Enter להתחלה, מדברים, Enter שוב לעצירה — ומקבלים תמלול + סיכום.

### עיבוד קובץ אודיו קיים
```powershell
.\.venv\Scripts\python.exe -m src.main "נתיב\לקובץ.wav"
.\.venv\Scripts\python.exe -m src.main --transcribe-only "קובץ.wav"   # רק תמלול
```

## מבנה הפרויקט
| קובץ | תפקיד |
|------|--------|
| `src/config.py` | הגדרות + טעינת `.env` |
| `src/transcribe.py` | תמלול מקומי (faster-whisper) |
| `src/summarize.py` | סיכום ומשימות (Gemini) |
| `src/record.py` | הקלטת מיקרופון + קול מערכת |
| `src/app_cli.py` | אפליקציית הקלטה חיה (CLI) |
| `src/main.py` | עיבוד קובץ קיים (CLI) |
| `src/storage.py` | קריאת סיכומים והקלטות שמורים |
| `src/export_word.py` | ייצוא סיכום ל-Word (RTL) |
| `src/watcher.py` | זיהוי פגישות + אייקון מגש |
| `src/detect.py` | זיהוי פגישה פעילה |

## הגדרות (`.env`)
- `WHISPER_MODEL` — `tiny`/`base`/`small`/`medium`/`large-v3` (כרגע `small`)
- `WHISPER_LANGUAGE` — `he` לעברית
- `GEMINI_MODEL` — מודל הסיכום
